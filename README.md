# StockAssistant
股票/基金/债券相关信息的协助应用。

开发原因主要是不想装太多app，比如集思录，蛋卷之类的，把他们部分数据集合到这个App上；同时也想用jetpack相关组件练练手。
lib类代码来自https://github.com/KunMinX/Jetpack-MVVM-Best-Practice。

注：此App中数据不构成任何投资建议。

债券部分：（数据来自集思录）
1.新债申购和上市相关的时间
2.筛选已经上市比较低估的可转债（lip师兄的思维逻辑）

股票股份：（数据来自集思录及理杏仁）
1.港股打新IPO的股票
2.个人筛选的一些后宫团股票（lip师兄的思维逻辑）

基金部分：（数据来自蛋卷）
1.钉大发车的基金信息
2.基金可以套利的部分

策略部分：（数据来自也大星球）
根据也大投资的逻辑链整理的投资策略
